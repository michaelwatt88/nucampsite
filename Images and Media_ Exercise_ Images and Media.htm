    Images and Media

    2. Exercise: Images and Media

    Click here if you wish to open this video in its own window.


    Objectives

    • Add images to the course project using Bootstrap image classes.
    • Learn to use the media object class to control layout of images with other elements such as text.


    Instructions

    Download and extract image files

    • Download the zip file below and extract it in your project folder. This should create a folder there named img.
    img.zip


    Add the website logo images

    • In index.html, go to the header element with the jumbotron class. Inside it, just below the row div and above the existing column div there, add the following column div element: 
    . . .
                <div class="col-4 col-sm-3 col-md-2 align-self-center">
                   <img src="img/logo.png" class="img-fluid" />
                </div>
. . .

    • You will immediately notice the logo being displayed in the jumbotron.
    • Add a smaller version of the logo to the website navigation bar as well. In index.html, go to the nav element, and just beneath the container div, add this code:

                <a class="navbar-brand" href="#"><img src="img/logo.png" height="30" width="30" /></a>

    • Go to the other pages and add the above code as well to add the logo to the jumbotron and the navbar. However, in the other pages, replace the  "#" with "index.html" because it is commonly expected UI behavior for the logo images on a webpage to lead back to the homepage when clicked.
    • Similarly, for all pages except index.html, surround the header image with anchor tags to cause the logo image to return the user to the homepage when clicked, as shown below:
                        <a href="index.html"><img src="img/logo.png" class="img-fluid" /></a>


    Add Media Objects

    • Update the second column in the first row-content div in index.html with this code:
    . . .
            <div class="col">
                <div class="media">
                    <img class="d-flex mr-3 img-thumbnail" src="img/breadcrumb-trail-thumb.jpg" alt="Breadcrumb Trail" />
                    <div class="media-body align-self-center">
                        <h3>Our Campsites</h3>
                        <p class="d-none d-sm-block">Explore our growing database of curated campsites and leave your own reviews!</p>
                    </div>
                </div>
            </div>
. . .

    • Similarly, update the second column in the third row-content div with this code:

    . . .
            <div class="col">
                <div class="media">
                    <img class="d-flex mr-3 img-thumbnail " src="img/bootstrap-thumb.png" alt="Bootstrap Outfitters">
                    <div class="media-body align-self-center">
                        <h3>Bootstrap Outfitters</h3>
        				<p class="d-none d-sm-block">Bootstrap Outfitters supplies you with gear you need at prices you can't beat.</p>
                    </div>
                </div>
            </div>
. . .


    Optional: Git

    • Commit the changes to your local repository with the message "Images and Media". Push the commit to your remote repository.


    Summary

    • In this lesson, you learned to add Bootstrap's responsive image classes to your project, and about using the Bootstrap media object for layout, to help position images around other elements properly.
